Compiled with gcc ver. 7.5.0 on Ubuntu Linux 18.04


To compile: gcc -pthread -o stacknode stacknode.c

To run: navigate to directory and execute command ./stacknode

Program creates 200 threads, for each thread 3 pop and 3 pull operations are executed on the stack over 500 iterations.

Printf Statements are enabled to print out each value that is pushed and popped.
